# Changelog

All notable changes to Marvel Rivals Instalocker will be documented in this file.

## [1.0.0] - 2025-01-19

### Added
- 🚀 Initial release of Marvel Rivals Instalocker
- 🎯 Support for all current Marvel Rivals heroes (33 total)
- 🖼️ Visual hero selection with category organization
- ⚡ Computer vision-based hero detection and locking
- 🎮 Clean GUI interface with tooltips and visual feedback
- 🛡️ Error handling and robust detection algorithms
- 📱 Support for 1920x1080 resolution (primary)
- 🔄 Threading to prevent GUI freezing during operations

### Hero Support
- **Duelists (17)**: Hawkeye, Black Panther, Black Widow, Iron Man, Iron Fist, Magik, Moon Knight, Namor, Psylocke, Scarlet Witch, Spider-Man, Squirrel Girl, Star-Lord, Storm, The Punisher, Hela, Winter Soldier, Wolverine
- **Strategists (7)**: Jeff the Land Shark, Adam Warlock, Loki, Cloak & Dagger, Luna Snow, Rocket Raccoon, Mantis  
- **Vanguards (8)**: Thor, Venom, Captain America, Doctor Strange, Groot, Hulk, Magneto, Peni Parker

### Technical Features
- Template matching with OpenCV for hero detection
- PyAutoGUI for automated clicking and screen interaction
- Tkinter-based GUI with image caching for performance
- Configurable detection confidence and timing parameters
- Cross-platform Python compatibility

### Documentation
- Comprehensive README with setup instructions
- Contributing guidelines for community development
- MIT License for open source usage
- Detailed code documentation and comments
